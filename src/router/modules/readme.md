## 路由配置说明：

- name 路由名称, 必须设置,且不能重名
- redirect 重定向地址, 访问这个路由时,自定进行重定向
- meta 路由元信息（路由附带扩展信息）
- meta.disabled 禁用整个菜单
- meta.title 菜单名称
- meta.icon 菜单图标
- meta.permissions 有权限的角色ID
- meta.keepAlive 缓存该路由
- meta.sort 排序越小越排前
